# Arcade M&L - Jogos em C

Bem-vindo ao Arcade M&L! Este projeto nasceu de um trabalho para a faculdade. 
Aqui, você encontrará três jogos distintos que testam desde seu conhecimento geral até sua capacidade estratégica.
Esse é o meu primeiro projeto em C. Ele contém jogos desenvolvidos para o console.
Fiz ele juntamente com a minha amiga Leticia.

## 🛠️O que eu aprendi

Desenvolver este Arcade foi uma grande escola de **Lógica de Programação**. Durante o processo, foquei em:

* **Programação Defensiva:** Implementação de verificações rigorosas para garantir que o jogo não "quebre" com entradas inválidas do usuário.
* **Gestão de Estados:** Controle preciso de turnos, níveis de fúria e condições de vitória/derrota através de vetores.
* **Mecânicas de Jogo:** Criação de uma lógica de "reviver" personagens baseada em transferência de valores entre índices de um array.

## 🎮 Os Jogos
1. **Pergunta e Resposta**: Um quiz de conhecimentos gerais.
2. **Cobra na Caixa**: Um jogo de sorte e estratégia para 2 jogadores.
3. **Gousmas War**: O jogo principal. Um RPG de estratégia por turnos onde você deve gerenciar a fúria das suas criaturas para derrotar o oponente.

## 🤖 Uso de Inteligência Artificial

Este projeto contou com o auxílio de **ferramentas de Inteligência Artificial** durante o seu ciclo de desenvolvimento. A IA foi utilizada de forma estratégica para:
* **Refinação de Lógica:** Apoio na identificação de *edge cases* (casos extremos) e bugs de lógica nas mecânicas de transferência de pontos.
* **Otimização de Código:** Sugestões de melhorias na legibilidade e estrutura das funções.

O uso da tecnologia serviu como um "copiloto", permitindo focar na arquitetura do jogo e na experiência do usuário, enquanto a ferramenta ajudava a validar a robustez do código.
