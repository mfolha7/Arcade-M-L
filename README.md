# Arcade M&L - Jogos em C

Este projeto contém três jogos desenvolvidos em linguagem C para o console.

## 🎮 Os Jogos
1. **Pergunta e Resposta**: Um quiz de conhecimentos gerais.
2. **Cobra na Caixa**: Um jogo de sorte e estratégia para 2 jogadores.
3. **Gousmas War**: O jogo principal. Um RPG de estratégia por turnos onde você deve gerenciar a fúria das suas criaturas para derrotar o oponente.

## 🛠️ Lógica de Programação
O projeto utiliza conceitos de:
- Vetores e Matrizes.
- Estruturas de controle (`switch`, `if/else`).
- Laços de repetição (`do-while`, `while`).
- Programação defensiva (validação de entradas do usuário).
- Manipulação de memória e ponteiros.

## 🚀 Como Executar
1. Tenha um compilador de C instalado (GCC, Code::Blocks ou Dev-C++).
2. Compile o arquivo `jogos.c`.
3. Execute o executável gerado.
